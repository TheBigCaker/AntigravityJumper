Option A — Same Network (LAN)

Find the desktop's local IP (run ipconfig on Windows). Edit config.json:

"relay": { "url": "ws://192.168.1.XXX:8765" }

Option B — Cloudflare Tunnel (Recommended, free)

Run on the desktop to get a public URL without opening ports:

winget install Cloudflare.cloudflared
cloudflared tunnel --url http://localhost:8765

It prints a URL like https://xxx.trycloudflare.com. Use it in config:

"relay": { "url": "wss://xxx.trycloudflare.com" }

Option C — ngrok

ngrok http 8765
# Use the wss:// URL it gives you

Relay Server (run on desktop)

cd codai_daemon
python relay_server.py
# Dashboard opens at http://localhost:8765/

Set RELAY_SECRET=yourpassword env var on both machines for auth.

Daemon (run on each machine)

python daemon.py --config config.json
# Laptop: set machine.id = "laptop"
# Desktop: set machine.id = "desktop"
